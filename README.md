# Ecommerce Website with Admin Panel

Welcome to the **Ecommerce Website with Admin Panel** project! This is a full-featured ecommerce platform built with modern technologies, featuring a user-facing storefront and a powerful admin panel for managing products, orders, users, and more.

## Demo

- **Live Site:** [Ecommerce Website](https://ecommercewebsite-with-admin-panel.vercel.app/)
- **Admin Panel:** [Admin Login](https://ecommercewebsite-with-admin-panel.vercel.app/admin/login)

---

## Features

### User Side
- Browse products by category, price, and popularity
- Product details, cart, and wishlist functionality
- Secure checkout and order management
- User authentication and profile management
- Responsive design for mobile and desktop

### Admin Panel
- Admin authentication system
- Add, edit, and delete products
- Manage categories and inventory
- View and process orders
- Manage users and analytics

## Getting Started

### Prerequisites

- Node.js (v16+)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/NailaFatima/Ecommercewebsite-with-AdminPanel.git
   cd Ecommercewebsite-with-AdminPanel
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables:
   - Create a `.env` file in the root directory
   - Add necessary variables (DB connection, JWT secret, etc.)

4. Run the development server:
   ```bash
   npm start
   ```
